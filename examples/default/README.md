# Default example

This example shows how to deploy the module in its simplest configuration.

```hcl
terraform {
  required_version = "~> 1.6"
  required_providers {}
}

module "regions" {
  source           = "../../"
  enable_telemetry = var.enable_telemetry
}

output "regions" {
  value = module.regions.regions
}
```
